| 科技感 | `circuitry, data streams, scan lines, UI elements, hexagonal grid` |

### Midjourney 参数说明

| 参数 | 说明 |
|------|------|
| `--ar 16:9` | 横向背景用，`--ar 3:4` 竖向角色立绘用 |
| `--style raw` | 减少 MJ 默认美化，更接近真实/电影感 |
| `--v 6.0` | 使用 MJ v6 模型，细节最佳 |
| `--stylize 500-850` | 数值越高艺术化越强，500 偏写实，850 偏风格化 |

### Stable Diffusion 推荐参数

| 参数 | 推荐值 |
|------|--------|
| 采样方法 | DPM++ 2M Karras / Euler a |
| 采样步数 | 28-40 |
| CFG Scale | 7-9 |
| 尺寸 | 1920×1080（背景）/ 768×1024（角色） |
| 模型推荐 | Realistic Vision / CyberRealistic / SDXL Base |
